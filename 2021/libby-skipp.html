Libby Skipp

Computer algorithms have the ability to ‘see’, they can detect colour and shapes within an image. This project book explores colour, and the way computers see it. The images in the publication have been sourced using three different websites, all of which use colour detection to produce search results. The print-on-demand book can be previewed on Blurb, and both the print book and ebook files can be downloaded for free via the Internet Archive. The project also has an Instagram account, @machinevisionandcolour, re-circulating some of the images from the book.
